WebWarriors Project Overview

Abstract

The internet, aside from all its benefits, has also introduced challenges affecting users who lack proper knowledge. WebWarriors is a video game designed to teach users how to handle various online threats through user-friendly methodologies. Inspired by the combat system found in classic RPGs, this game aims to educate users in a fun and interactive manner.

Introduction

The Internet has transformed society, offering unparalleled information and communication. However, it comes with issues such as phishing, identity theft, and cyberbullying. WebWarriors addresses these challenges using game-based learning to teach users effective online safety practices.

Main Objectives

  • Teach users to identify and respond to common online threats.
  • Promote safe online behavior.
  • Increase user engagement through interactive learning.
  • Ensure broad accessibility and ease of use.

Technological Proposal

The solution involves developing a 2D game where players interact with the world, jump on platforms, and engage in combat scenarios. The game includes a randomness component ensuring diverse gameplay experiences, with multiple character and enemy designs integrated cohesively.

Functional Requirements

  • Allow the player to move and jump in a 2D world.
  • Recognize click interactions.
  • Play music and sound effects in real-time.
  • Display visual assets such as characters and backgrounds.
  • Implement a randomness component for unique gameplay.

Non-Functional Requirements

  • Clean, well-commented, and easy-to-read code.
  • Error handling to minimize critical failures.
  • Intuitive interface with instructional messages.
  • Performance optimization using tailored data structures.
